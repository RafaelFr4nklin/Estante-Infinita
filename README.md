# 📚 Estante Infinita

Loja virtual de cards Pokémon e quadrinhos desenvolvida com HTML, CSS e JavaScript puro.

## 🔗 Acesse o site
[estante-infinita](https://rafaelfr4nklin.github.io/Estante-Infinita/HTML/index.html)

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript
- localStorage para persistência do carrinho

## ⚙️ Funcionalidades
- Catálogo de cartas Pokémon e quadrinhos
- Página de detalhes do produto com descrição
- Carrinho com controle de quantidade por item
- Checkbox para selecionar itens a comprar
- Cálculo automático do total com exibição parcela a parcela
- Remoção de itens do carrinho
- Redirecionamento automático para a loja
